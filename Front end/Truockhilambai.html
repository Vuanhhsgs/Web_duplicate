Usage Instructions

This website provides a curated collection of geometry problems. For the best experience, please download and install Geometer's Sketchpad from the official site: keycurriculum.com.

After installing, the app will prompt for license information. To use it with the settings provided here, enter:
License name: Pythagoras Single User License 1863 KJUJAM
Authorization code: BDFBF968

Each problem on this site follows the format shown below:

Click "My Solution" to view my solution.

Click "Try It Yourself" to download a .gsp file. Open that file with Geometer's Sketchpad and work from the prepared figure.

On the top left corner, there's a button "SHOW TECHNIQUES USED", click on it to show every techniques that are used for every problem.

Click on it again to hide all techniques that are used for every problem
